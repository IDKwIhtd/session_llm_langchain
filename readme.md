## How to use

```bash
# Virtual Environment
python -m venv lang-venv

# macOS
source lang-venv/bin/activate

# Windows
lang-venv\Scripts\activate

# Install Requirements
pip install -r requirements.txt

# Run
python 001_llm_start.py
```
